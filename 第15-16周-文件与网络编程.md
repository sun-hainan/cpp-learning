## 文件操作：存档功能的底层原理

### 为什么要存档？

你写传奇脚本时，肯定做过玩家数据存储：
```lua
-- Lua: 把玩家数据写入文件
file = io.open("player_" .. playerId .. ".dat", "w")
file:write(playerName .. "\n")
file:write(playerLevel .. "\n")
file:close()
```

C++ 的文件操作和 Lua 的思路完全一样，只是语法不同。

### 基本文件读写

```cpp
#include <fstream>
using namespace std;

// 写入文件
void save() {
    ofstream out("save.txt");  // ofstream = output file stream
    out << "战士 30 100 50" << endl;
    out << "法师 25 80 100" << endl;
    out.close();  // 关闭文件
    cout << "存档成功" << endl;
}

// 读取文件
void load() {
    ifstream in("save.txt");   // ifstream = input file stream
    if (!in) {                  // 检查文件是否存在
        cout << "存档文件不存在" << endl;
        return;
    }

    string name;
    int level, hp, mp;

    while (in >> name >> level >> hp >> mp) {  // 逐行读取
        cout << name << " 等级" << level
             << " HP" << hp << " MP" << mp << endl;
    }

    in.close();
}
```

**等价的 Lua 写法对比：**

```lua
-- Lua: 写
local f = io.open("save.txt", "w")
f:write("战士 30 100 50\n")
f:close()

-- Lua: 读
local f = io.open("save.txt", "r")
for line in f:lines() do
    print(line)
end
f:close()
```

### 练习：玩家存档系统

```cpp
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

struct Player {
    string name;
    int level;
    int hp;
    int mp;
};

void savePlayers(Player* players, int count) {
    ofstream out("players.txt");
    for (int i = 0; i < count; i++) {
        out << players[i].name << " "
            << players[i].level << " "
            << players[i].hp << " "
            << players[i].mp << endl;
    }
    out.close();
    cout << "存档完成，共" << count << "个玩家" << endl;
}

void loadPlayers(Player*& players, int& count) {
    ifstream in("players.txt");
    if (!in) {
        cout << "无存档" << endl;
        return;
    }

    count = 0;
    players = new Player[100];  // 动态分配

    while (in >> players[count].name
                >> players[count].level
                >> players[count].hp
                >> players[count].mp) {
        count++;
    }
    in.close();
    cout << "读档完成，共" << count << "个玩家" << endl;
}

int main() {
    Player p[3] = {
        {"战士", 30, 100, 50},
        {"法师", 25, 80, 100},
        {"道士", 20, 60, 80}
    };

    savePlayers(p, 3);

    Player* loaded;
    int count;
    loadPlayers(loaded, count);

    for (int i = 0; i < count; i++) {
        cout << loaded[i].name << " 等级" << loaded[i].level << endl;
    }

    delete[] loaded;
    return 0;
}
```

## socket 编程：网络通信基础

### 你在 Lua 里写过的 TCP 通信

```lua
-- Lua: TCP 服务器端
local server = require("socket").bind("*", 8888)
while true do
    local client = server:accept()
    local data = client:receive()
    -- 处理数据
    client:send("收到\n")
end
```

C++ 的 socket 编程更底层，但思路完全一样：
- 服务器绑定端口、监听连接
- 客户端连接服务器
- 双方收发数据

### Windows socket 初始化

```cpp
#include <winsock2.h>
#pragma comment(lib, "ws2_32.lib")

// 初始化 Winsock
WSADATA wsa;
WSAStartup(MAKEWORD(2, 2), &wsa);  // 启动 Winsock
// ... 使用 socket ...
WSACleanup();  // 关闭 Winsock
```

### TCP 客户端

```cpp
#include <iostream>
#include <winsock2.h>
#pragma comment(lib, "ws2_32.lib")

using namespace std;

int main() {
    // 1. 初始化 Winsock
    WSADATA wsa;
    WSAStartup(MAKEWORD(2, 2), &wsa);

    // 2. 创建 socket
    SOCKET clientSocket = socket(AF_INET, SOCK_STREAM, 0);

    // 3. 连接服务器
    sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;         // IPv4
    serverAddr.sin_port = htons(8888);        // 端口号
    serverAddr.sin_addr.s_addr = inet_addr("127.0.0.1");  // 本机IP

    connect(clientSocket, (sockaddr*)&serverAddr, sizeof(serverAddr));
    cout << "已连接服务器" << endl;

    // 4. 收发数据
    const char* msg = "Hello Server!";
    send(clientSocket, msg, strlen(msg), 0);

    char buffer[1024] = {0};
    int len = recv(clientSocket, buffer, sizeof(buffer), 0);
    cout << "服务器回复: " << buffer << endl;

    // 5. 关闭
    closesocket(clientSocket);
    WSACleanup();

    return 0;
}
```

### TCP 服务器端

```cpp
#include <iostream>
#include <winsock2.h>
#pragma comment(lib, "ws2_32.lib")
using namespace std;

int main() {
    WSADATA wsa;
    WSAStartup(MAKEWORD(2, 2), &wsa);

    // 1. 创建监听 socket
    SOCKET listenSocket = socket(AF_INET, SOCK_STREAM, 0);

    // 2. 绑定端口
    sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;
    serverAddr.sin_port = htons(8888);
    serverAddr.sin_addr.s_addr = INADDR_ANY;  // 接受任何IP的连接

    bind(listenSocket, (sockaddr*)&serverAddr, sizeof(serverAddr));

    // 3. 监听
    listen(listenSocket, 5);  // 最多5个连接排队
    cout << "服务器启动，监听端口 8888..." << endl;

    // 4. 接受连接
    sockaddr_in clientAddr;
    int addrLen = sizeof(clientAddr);
    SOCKET clientSocket = accept(listenSocket,
                                  (sockaddr*)&clientAddr,
                                  &addrLen);
    cout << "客户端连接！" << endl;

    // 5. 收发数据
    char buffer[1024];
    int len = recv(clientSocket, buffer, sizeof(buffer), 0);
    buffer[len] = '\0';
    cout << "收到: " << buffer << endl;

    const char* reply = "消息已收到";
    send(clientSocket, reply, strlen(reply), 0);

    // 6. 关闭
    closesocket(clientSocket);
    closesocket(listenSocket);
    WSACleanup();

    return 0;
}
```

### socket 关键函数解释

| 函数 | 作用 |
|------|------|
| `socket()` | 创建 socket |
| `bind()` | 把 socket 绑定到 IP + 端口 |
| `listen()` | 开始监听连接 |
| `accept()` | 接受一个连接请求 |
| `connect()` | 客户端连接服务器 |
| `send()` | 发送数据 |
| `recv()` | 接收数据 |
| `htons()` | 把端口号转成网络字节序 |
| `inet_addr()` | 把 IP 字符串转成二进制 |

### 和 Lua socket 的对应关系

```lua
-- Lua 服务器
server = socket.bind("*", 8888)
listen(server, 5)
client = server:accept()
data = client:receive()
client:send("reply")
```

```cpp
// C++ 服务器
listenSocket = socket(...)
bind(listenSocket, &addr)
listen(listenSocket, 5)
clientSocket = accept(listenSocket, ...)
// 收到数据后回复
send(clientSocket, "reply", ...)
```

**思路完全一样，只是 C++ 要手动管理每个步骤。**

## 本周总结

| 概念 | 记住一句话 |
|------|-----------|
| ofstream / ifstream | 文件输出/输入流 |
