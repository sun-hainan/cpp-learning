# 第7-8周：阶段项目——通讯录管理系统

---

## 项目目标

用前6周学的内容，做一个完整的通讯录程序：

```
===== 通讯录管理 =====
1. 添加联系人
2. 删除联系人
3. 查找联系人
4. 修改联系人
5. 显示所有联系人
0. 退出

请选择: _
```

**涉及的技术点：**
- struct 定义联系人
- vector 存储联系人列表
- map 加速查找
- if/else 菜单选择
- while 循环菜单
- 函数封装

## 第三步：添加联系人

```cpp
void addContact(vector<Contact>& contacts) {
    Contact c;
    cout << "姓名: ";
    cin >> c.name;
    cout << "电话: ";
    cin >> c.phone;
    cout << "邮箱: ";
    cin >> c.email;

    contacts.push_back(c);
    cout << "添加成功！" << endl;
}
```

**注意：`vector<Contact>&` 里的 `&` 是引用——让函数直接修改外面的 contacts，而不是复制一份。**

---

## 第四步：查找联系人

```cpp
void findContact(const vector<Contact>& contacts) {
    string name;
    cout << "输入要查找的姓名: ";
    cin >> name;

    bool found = false;
    for (const Contact& c : contacts) {
        if (c.name == name) {
            cout << "找到: " << c.name
                 << " | 电话: " << c.phone
                 << " | 邮箱: " << c.email << endl;
            found = true;
            break;
        }
    }

    if (!found) {
        cout << "未找到: " << name << endl;
    }
}
```

---

## 第五步：删除联系人

```cpp
void deleteContact(vector<Contact>& contacts) {
    string name;
    cout << "输入要删除的姓名: ";
    cin >> name;

    for (auto it = contacts.begin(); it != contacts.end(); it++) {
        if (it->name == name) {
            contacts.erase(it);
            cout << "删除成功！" << endl;
            return;
        }
    }

    cout << "未找到: " << name << endl;
}
```

---

## 第六步：显示所有

```cpp
void showAll(const vector<Contact>& contacts) {
    if (contacts.empty()) {
        cout << "通讯录为空" << endl;
        return;
    }

    cout << "\n===== 通讯录列表 =====" << endl;
    for (const Contact& c : contacts) {
        cout << c.name << " | " << c.phone << " | " << c.email << endl;
    }
    cout << "共 " << contacts.size() << " 个联系人" << endl;
}
```

## 完整代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

struct Contact {
    string name;
    string phone;
    string email;
};

void showMenu() {
    cout << "===== 通讯录管理 =====" << endl;
    cout << "1. 添加联系人" << endl;
    cout << "2. 删除联系人" << endl;
    cout << "3. 查找联系人" << endl;
    cout << "4. 修改联系人" << endl;
    cout << "5. 显示所有联系人" << endl;
    cout << "0. 退出" << endl;
    cout << "请选择: ";
}

void addContact(vector<Contact>& contacts) {
    Contact c;
    cout << "姓名: ";
    cin >> c.name;
    cout << "电话: ";
    cin >> c.phone;
    cout << "邮箱: ";
    cin >> c.email;
    contacts.push_back(c);
    cout << "添加成功！" << endl;
}

void findContact(const vector<Contact>& contacts) {
    string name;
    cout << "输入要查找的姓名: ";
    cin >> name;
    for (const Contact& c : contacts) {
        if (c.name == name) {
            cout << "找到: " << c.name
                 << " | 电话: " << c.phone
                 << " | 邮箱: " << c.email << endl;
            return;
        }
    }
    cout << "未找到: " << name << endl;
}

void deleteContact(vector<Contact>& contacts) {
    string name;
    cout << "输入要删除的姓名: ";
    cin >> name;
    for (auto it = contacts.begin(); it != contacts.end(); it++) {
        if (it->name == name) {
            contacts.erase(it);
            cout << "删除成功！" << endl;
            return;
        }
    }
    cout << "未找到: " << name << endl;
}

void showAll(const vector<Contact>& contacts) {
    if (contacts.empty()) {
        cout << "通讯录为空" << endl;
        return;
    }
    cout << "\n===== 通讯录列表 =====" << endl;
    for (const Contact& c : contacts) {
        cout << c.name << " | " << c.phone << " | " << c.email << endl;
    }
    cout << "共 " << contacts.size() << " 个联系人" << endl;
}

int main() {
    vector<Contact> contacts;
    int choice;

    while (true) {
        showMenu();
        cin >> choice;

        switch (choice) {
            case 1: addContact(contacts);    break;
            case 2: deleteContact(contacts);  break;
            case 3: findContact(contacts);    break;
            case 5: showAll(contacts);        break;
            case 0:
                cout << "再见！" << endl;
                return 0;
            default:
                cout << "无效选择" << endl;
        }
        cout << endl;
    }
}
```
