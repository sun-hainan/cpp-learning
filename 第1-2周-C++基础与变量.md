# 第1周：C++ 是什么？和 Lua 有什么区别？

---

## 先看一个生活中的例子：厨房和厨师

**Lua 像是：**
> 一键式傻瓜厨房。你按一个按钮，厨房自动把所有食材准备好、切好、炒好。你只需要说"做宫保鸡丁"，它就给你端出来。

**C++ 像是：**
> 一个空的厨房，从零开始。你要自己决定用什么锅、用什么火候、食材切成什么形状。你拥有完全的掌控权，但也要对每一个细节负责。

两种厨房都能做出好吃的菜，只是方式不同。

---

## 什么是 C++？

C++ 是由 Bjarne Stroustrup（比雅尼·斯特劳斯特鲁普）在 1980 年代创建的编程语言。"++"意味着"在 C 的基础上加一"，但实际上 C++ 早就不只是"C 的升级版"了——它是一门**静态编译型、多范式、运行速度极快**的语言。

**静态编译型**的意思是：
```
代码写完后，要经过一个"翻译"过程，才能变成电脑能读懂的语言
这个"翻译"过程叫做"编译"
编译完成后，程序可以独立运行，不需要额外的翻译器
```

**和 Lua 的根本区别：**

| 对比项 | Lua | C++ |
|--------|-----|-----|
| 运行方式 | 脚本语言，运行时逐行解释执行 | 编译型语言，先全部翻译成机器码再运行 |
| 速度 | 较慢（需要虚拟机） | 极快（直接运行机器码） |
| 内存管理 | 自动垃圾回收（GC） | 手动管理（new/delete） |
| 类型检查 | 动态类型（变量类型自动推断） | 静态类型（变量类型必须声明） |
| 指针 | 没有 | 有（直接操作内存地址） |
| 编译/运行 | 不需要编译，直接运行 | 需要先编译成 .exe 再运行 |
| 学习门槛 | 低，上手快 | 较高，但掌握后威力大 |

## 为什么游戏公司喜欢 C++？

你写传奇脚本用的是 Lua，但传奇引擎本身是 C++ 写的。游戏行业选择 C++ 的原因：

1. **速度**：游戏需要实时渲染和物理计算，C++ 比 Lua 快 10-100 倍
2. **内存控制**：游戏服务器需要精细管理大量玩家数据，C++ 可以精确控制每一块内存
3. **跨平台**：C++ 编译后可以在 Windows、Linux、Mac、Android 等平台运行
4. ** Unreal / Unity 底层**：主流游戏引擎的核心都是 C++

## 你的优势：用 Lua 的经验来学 C++

Lua 里你写过的东西，C++ 都能实现，只是方式不同：

```
Lua 里的概念          C++ 里对应的概念
─────────────────────────────────────
变量                  变量（但要声明类型）
if 条件判断           if 条件判断（几乎一样）
for 循环              for 循环（几乎一样）
table                 vector + map + struct
function              函数（但参数要声明类型）
while 循环            while 循环（几乎一样）
nil（空值）           nullptr（空指针）
字符串操作            string 类
```

**最大的区别只有一个：指针。** Lua 没有指针，C++ 有。掌握了指针，就掌握了 C++ 的一半。

## 第一个 C++ 程序：Hello World

**在 Lua 里，打印一句话是这样：**
```lua
print("Hello, World!")
```

**在 C++ 里，打印同样一句话是这样：**
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

现在逐行解释，每一行是什么意思：

```cpp
#include <iostream>   // 第1行：引入输入输出工具箱
                     // 等于 Lua 里的 require "io"
                     // iostream = "输入输出流"（Input Output Stream）
                     // 有了它，我们才能用 cout 来打印内容

using namespace std;  // 第2行：使用标准命名空间
                     // 把 std 里的工具直接拿过来用
                     // 就像 Lua 里写 local print = print

int main() {          // 第3行：主函数，所有程序的入口
                     // 电脑从这一行开始执行
                     // int = 这个函数的返回值类型是整数
                     // Lua 里没有这个概念，Lua 函数可以返回任何类型

    cout << "Hello, World!" << endl;  // 第4行：打印内容
                     // cout = console output（控制台输出）
                     // <<  是"插入运算符"，把内容送进屏幕
                     // endl = end line，换行
                     // 等价于 Lua 的 print("Hello, World!")

    return 0;         // 第5行：返回0表示程序正常结束
                     // 0 = 正常退出，非0 = 有错误
}
```

**保存为 hello.cpp**（注意后缀是 .cpp，不是 .lua）

## 编译：把你的代码变成可执行文件

Lua 是**解释型语言**，写完就能直接运行：
```bash
lua hello.lua
```

C++ 是**编译型语言**，要先"翻译"成机器语言，再运行：
```bash
# 编译（翻译成 hello.exe）
cl /EHsc /std:c++17 hello.cpp /Fe:hello.exe

# 运行
hello.exe
```

**什么是编译？**

想象你写了一封中文信，要寄给一个不懂中文的外国人。你需要先找人把中文翻译成英文，这封信才能被对方读懂。

```
源代码（hello.cpp） → 编译 → 机器语言（hello.exe） → 运行
     中文信           翻译           英文信            外国人读
```

**编译报错常见原因：**
- 拼写错误：`cout` 写成 `cout`（少一个字母）
- 中英文符号混用：中文的引号""不等于英文的 ""
- 少了分号：C++ 每条语句必须以分号结尾 `;`
- 少了头文件：用了 `cout` 但没有 `#include <iostream>`

## 变量：给数据起个名字

**Lua 里，变量直接赋值就行：**
```lua
name = "张三"
age = 28
height = 172.5
```

**C++ 里，变量必须先"声明类型"：**
```cpp
string name = "张三";   // 字符串类型（Lua 里没有这个概念）
int age = 28;           // 整数类型
double height = 172.5;  // 浮点数类型
```

**为什么要声明类型？**

Lua 是"动态类型"——变量可以随时改变类型：
```lua
x = 10        -- x 是数字
x = "hello"  -- x 变成字符串，完全合法
```

C++ 是"静态类型"——变量声明后类型就固定了：
```cpp
int x = 10;   // x 是整数
x = "hello"; // 错误！不能把字符串赋给整数变量
```

**类型的作用：**
- 告诉计算机要分配多少内存（`int` 占用4字节，`double` 占用8字节）
- 提前发现错误，而不是运行到一半突然崩溃
- 让编译器能优化代码，运行更快

---

## C++ 常用数据类型速查表

| 类型 | 含义 | 举例 | 占用内存 |
|------|------|------|---------|
| `int` | 整数 | `28`, `-100`, `0` | 4字节 |
| `double` | 浮点数（小数） | `172.5`, `3.14` | 8字节 |
| `float` | 单精度浮点 | `172.5f` | 4字节 |
| `char` | 字符 | `'A'`, `'好'` | 1字节 |
| `bool` | 布尔（真假） | `true`, `false` | 1字节 |
| `string` | 字符串 | `"张三"` | 动态大小 |

**和 Lua 的对应关系：**

```
Lua 类型              C++ 类型
──────────────────────────────
nil                   无对应（C++ 没有空类型初始值）
number（整数）        int
number（小数）        double
string               string（需引入头文件）
boolean              bool
table                多种组合：数组+struct+map
function             函数（语法略有不同）
```

## 练习1：你的第一个 C++ 程序

**题目：**
创建一个程序，输出你的个人信息：
```
姓名: 张三
年龄: 28
是否学生: 0
```

**参考代码：**
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name = "张三";
    int age = 28;
    bool isStudent = false;

    cout << "姓名: " << name << endl;
    cout << "年龄: " << age << endl;
    cout << "是否学生: " << isStudent << endl;

    return 0;
}
```

**关键点：**
- `cout << "姓名: " << name << endl;` 里 `<<` 可以连续使用，把多段内容拼接输出
- `bool` 类型的 `false` 打印出来是 `0`，`true` 打印出来是 `1`

## 常见错误：分号和引号

**错误1：忘了分号**
```cpp
cout << "Hello"    // 错误！每条语句必须以 ; 结尾
cout << "World"   // 两个错误
```

**错误2：中文引号**
```cpp
cout << "Hello";   // 正确：英文引号 ""
cout << "Hello";   // 错误：中文引号 ""
```

**错误3：拼写错误**
```cpp
string name = "张三";  // 正确
strng name = "张三";  // 错误：string 拼写错了
```

## 本周总结

| 概念 | 记住一句话 |
|------|-----------|
| C++ 是什么 | 编译型静态语言，速度极快，完全掌控硬件 |
