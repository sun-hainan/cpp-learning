## 从一个场景开始

你在传奇脚本里肯定写过类似这样的代码：

```lua
-- Lua: 存放5个玩家的等级
playerLevels = {15, 28, 33, 12, 45}
print(playerLevels[1])  -- 输出第一个：15
```

**问题来了：**
- Lua 的 table 可以存任意类型、任意数量
- C++ 的数组要事先定好大小，存满了怎么办？
- Lua 的索引从 1 开始，C++ 从几开始？

## C++ 数组基础

### 声明和初始化

```cpp
int levels[5];              // 声明一个能存5个int的数组，值是随机的
int levels[5] = {15, 28, 33, 12, 45};  // 声明时直接赋值
int levels[] = {15, 28, 33, 12, 45};   // 不写长度，编译器自动数
```

**关键点：索引从 0 开始，不是 1！**

```cpp
int levels[5] = {15, 28, 33, 12, 45};

cout << levels[0] << endl;  // 第1个：15
cout << levels[1] << endl;  // 第2个：28
cout << levels[4] << endl;  // 第5个：45
// levels[5]   ← 错误！超出范围！
```

**和 Lua 的对比：**

```lua
-- Lua
levels = {15, 28, 33, 12, 45}
print(levels[1])  -- Lua索引从1开始，输出15
print(levels[5])  -- 输出45
```

```
C++:  levels[0] = 第1个元素
Lua:  levels[1] = 第1个元素
```

**记忆口诀：C++ 的数组是从 0 开始数的，用 index 减去 1 就是 Lua 的习惯。**

### 遍历数组

**方法1：普通 for 循环**
```cpp
int levels[5] = {15, 28, 33, 12, 45};

for (int i = 0; i < 5; i++) {
    cout << "玩家" << (i+1) << "等级:" << levels[i] << endl;
}
```

**方法2：范围 for（c++11以后）**
```cpp
int levels[5] = {15, 28, 33, 12, 45};

for (int level : levels) {
    cout << level << " ";
}
```

**等价的 Lua 写法：**
```lua
levels = {15, 28, 33, 12, 45}
for i, level in ipairs(levels) do
    print("玩家" .. i .. "等级:" .. level)
end
```

### 数组的大小必须固定

这是 C++ 数组和 Lua table 的核心区别：

```lua
-- Lua：数组大小可以随时变
arr = {1, 2, 3}
table.insert(arr, 4)  -- 现在有4个元素
table.insert(arr, 5)  -- 现在有5个元素
-- Lua 的 table 是"动态"的，想放多少放多少
```

```cpp
// C++：数组大小必须固定，编译时就知道
int arr[5];  // 只能存5个，编译时就定死了
arr[5] = 6;  // 错误！数组越界！
```

**什么时候用 C++ 数组？什么时候用 vector（C++ 的动态数组）？**

| 场景 | 用什么 |
|------|--------|
| 数据量固定（如一周有7天） | `int week[7]` |
| 数据量运行时才知道（如玩家数量） | `vector<int> players` |
| 需要动态增删 | `vector`（后面会讲） |

## 字符串：字符数组的特殊形式

C++ 没有像 Lua 那样的原生字符串类型，`string` 是一个类（后面会讲），但底层是字符数组。

**字符数组：**
```cpp
char name[6] = {'L', 'u', 'a', 'n', 'x', '\0'};  // \0 是字符串结束标志
// 等价于：
char name[] = "Luanx";
```

**为什么要加 `\0`？**

```
内存中的样子：
| L | u | a | n | x | \0 |
0  1  2  3  4  5   6

\0 告诉 cout："到这里字符串就结束了"
```

**常用操作：**
```cpp
#include <cstring>  // 字符串操作函数

char s1[20] = "Hello";
char s2[20] = "World";

strlen(s1);      // 返回长度：5
strcmp(s1, s2);  // 比较两个字符串：相等返回0
strcpy(s1, s2);  // 把 s2 复制到 s1
strcat(s1, s2);  // 把 s2 接到 s1 后面
```

**但更推荐用 string 类：**
```cpp
#include <string>

string s1 = "Hello";
string s2 = "World";

s1 + s2;           // 拼接："HelloWorld"
s1.length();       // 长度：5
s1 == s2;          // 比较：直接用 ==
s1[0];             // 访问字符：'H'
s1.substr(0, 3);   // 截取："Hel"
```

**和 Lua 的对比：**

```lua
s1 = "Hello"
s2 = "World"
s3 = s1 .. s2  -- Lua 用 .. 拼接字符串
print(#s1)     -- Lua 用 # 取长度
```

### 练习1：数组实战

**题目：** 输入5个整数，存到数组，输出最大值和最小值

```cpp
#include <iostream>
#include <climits>  // INT_MIN, INT_MAX
using namespace std;

int main() {
    int arr[5];
    cout << "输入5个整数:" << endl;

    for (int i = 0; i < 5; i++) {
        cin >> arr[i];
    }

    int maxVal = INT_MIN;  // 初始化为最小可能的整数
    int minVal = INT_MAX;  // 初始化为最大可能的整数

    for (int i = 0; i < 5; i++) {
        if (arr[i] > maxVal) maxVal = arr[i];
        if (arr[i] < minVal) minVal = arr[i];
    }

    cout << "最大值:" << maxVal << endl;
    cout << "最小值:" << minVal << endl;

    return 0;
}
```

## 二维数组：表格数据

**场景：** 3个班，每班5个人的成绩

```cpp
// 3行5列的表格
int scores[3][5] = {
    {85, 90, 78, 92, 88},   // 班级1的成绩
    {72, 81, 95, 67, 79},   // 班级2的成绩
    {91, 88, 76, 85, 93}    // 班级3的成绩
};

cout << scores[0][0] << endl;  // 第1行第1列：85
cout << scores[2][4] << endl;  // 第3行第5列：93
```

**遍历二维数组：**
```cpp
for (int i = 0; i < 3; i++) {          // 外层：行
    for (int j = 0; j < 5; j++) {      // 内层：列
        cout << scores[i][j] << " ";
    }
    cout << endl;
}
```

**等价的 Lua 写法：**
```lua
scores = {
    {85, 90, 78, 92, 88},
    {72, 81, 95, 67, 79},
    {91, 88, 76, 85, 93}
}

print(scores[1][1])  -- Lua 索引从1开始：85
```

## 本周总结

| 概念 | 记住一句话 |
|------|-----------|
| 数组声明 | `类型 数组名[长度];` |
| 索引从0开始 | C++ 第一个元素是 `arr[0]`，不是 `arr[1]` |
| 数组大小固定 | 编译时确定，vector 才是动态的 |
| string 类 | 比字符数组更好用，直接 `+` 拼接、`==` 比较 |
| 二维数组 | `类型 名字[行][列];`，像表格 |
